# ProjetIA
TP dans le cadre du cours de Concepts et technologies IA

L'objectif du projet est de construire un modèle qui sera capable de prédire la démission d'un employé.
Nous entraînerons notre modèle à partir des données d'entraînements issues de la plateforme Kaggle, dont voici le lien : https://www.kaggle.com/datasets/ziya07/employee-churn-data
